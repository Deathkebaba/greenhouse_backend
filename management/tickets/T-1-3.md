**c) Implement `_delta_messages` query:**
```python
def _delta_messages(
    conn: sqlite3.Connection,
    player_id: int,
    since_turn: int,
) -> list[GameStateMessage]:
    rows = conn.execute(
        f"""
        SELECT {kw.message.SENDER},{kw.message.RECEIVER},
               {kw.message.TURN},{kw.message.MESSAGE}
        FROM {kw.message.TABLE_NAME}
        WHERE ({kw.message.SENDER} = ? OR {kw.message.RECEIVER} = ?)
          AND {_LAST_UPDATED_TURN_COLUMN} > ?
        """,
        (player_id, player_id, since_turn),
    ).fetchall()
    return [
        GameStateMessage(sender=r[0], receiver=r[1], turn=r[2], message=r[3])
        for r in rows
    ]
```

**d) Wire into delta response construction** (in the delta endpoint handler,
around line 2030+ of routes.py), add:
```python

```

```typescript
async function upsertMessage(
  db: SqliteWasmLifecycle,
  gameId: number | string,
  msg: DeltaSyncChanges["messages"][number]
): Promise<void> {
  await db.exec({
    sql: `INSERT OR REPLACE INTO ${SQLITE_CACHE_SCHEMA_TABLES.messages}
          (game_id, sender, receiver, turn, message)
          VALUES (?, ?, ?, ?, ?)`,
    bind: [gameId, msg.sender, msg.receiver, msg.turn, msg.message],
  });
}
```

**c) Wire into delta-apply transaction:**
In the delta-apply hard-reset block, add message table clearing:

In the delta-apply upsert loop, add:
```typescript

}
```
