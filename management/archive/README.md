# Archive

This directory stores archived planning and ticketing artifacts.

- `legacy-tickets/` stores historical ticket artifacts from the pre-plan ticketing system.
- `plans/` stores archived plan detail files and `plans/index.yaml` stores their archived plan index entries.
- `tickets/` stores archived ticket detail files and `tickets/index.yaml` stores their archived ticket index entries.

Cleaner is the only agent that should move current plan and ticket artifacts into this archive.
Do not modify archived artifacts except for explicit migration or audit tasks.
